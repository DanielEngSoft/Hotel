Na pasta que criou o arquivo .ui execultar no terminal o comando:

    pyside6-uic arquivo.ui -o arquivo.py

Depois de criar um arquivo .py com o seguinte codigo:

    from PySide6 import QtWidgets
    from arquivo import Ui_MainWindow  # Substitua "arquivo" pelo nome do seu arquivo

    class MyApp(QtWidgets.QMainWindow, Ui_MainWindow):
        def __init__(self):
            super().__init__()
            self.setupUi(self)

    if __name__ == "__main__":
        app = QtWidgets.QApplication([])
        window = MyApp()
        window.show()
        app.exec_()
